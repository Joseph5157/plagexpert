# PlagExpert Website Instructions

We are building a real multi-page website for PlagExpert using Nuxt 3, Vue 3, and Tailwind CSS.

Design direction:
- Clean white background
- Pink/red primary CTA color
- Soft pink section backgrounds
- Dark navy footer and confidentiality section
- Rounded cards
- Mobile-first responsive layout
- Professional academic service landing page style

Main homepage sections:
1. SiteHeader
2. HeroSection
3. TrustStrip
4. TrustSection
5. HowItWorksSection
6. ServicesPreview
7. StatsSection
8. TestimonialSection
9. AudienceSection
10. PricingPreview
11. FaqPreview
12. ConfidentialSection
13. FinalCtaSection
14. SiteFooter

Main pages for phase 1:
- Home page
- Services page
- Pricing page
- FAQ page
- Contact page
- Privacy Policy
- Terms of Service

Important:
- Build homepage first.
- Use reusable Vue components.
- Keep data in arrays when possible.
- Use Tailwind CSS only for styling.
- Do not add backend, login, payment, or database yet.
- WhatsApp buttons should be simple links for now.
- Avoid unsafe claims like guaranteed university match or official access unless content is later verified.
